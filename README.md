# 🛠️ Tools Collection

業務で使用する便利なツール集です。各ツールはブラウザで動作し、インストール不要で使用できます。

## 📚 ツール一覧

### 🎁 メッセージカード生成システム
[![デモ](https://img.shields.io/badge/デモ-View-blue)](https://yanagawatsubasa.github.io/tools/message-card-generator/)

Illustrator不要で、ブラウザだけでメッセージカードを作成・PDF出力できるツールです。

- **パス**: [`/message-card-generator`](./message-card-generator)
- **主な機能**:
  - テキスト入力でメッセージカード作成
  - リアルタイムプレビュー
  - 高品質PDF出力（ハガキサイズ）
  - 完全ブラウザベース（サーバー不要）
- **使い方**: [詳細はこちら](./message-card-generator/README.md)
- **デモページ**: https://yanagawatsubasa.github.io/tools/message-card-generator/

---

## 🚀 使い方

各ツールのディレクトリに移動し、`index.html` を開くだけで使用できます。

### ローカルで使用
```bash
# リポジトリをクローン
git clone https://github.com/yanagawatsubasa/tools.git
cd tools

# 使いたいツールのindex.htmlを開く
open message-card-generator/index.html  # Mac
# または
start message-card-generator/index.html # Windows
```

### オンラインで使用
GitHub Pagesで公開されているため、以下のURLから直接アクセスできます：
- https://yanagawatsubasa.github.io/tools/message-card-generator/
